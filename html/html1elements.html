Unit 2: HTML Elements

DOCTYPE

DOCTYPE tells the browser what type of document it is.

E.g. <!DOCTYPE html>
E.g. <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

html

Root element of an HTML page and contains all other elements.

E.g. <html>

head

Container for the metadata of the page, i.e. information that is not directly part of the page. about the page.

E.g. <head>

body

Container for the contents of the page.

E.g. <body>

Title

A title is a tag to describe a webpage

<title> "skitheslip" </title>

h1

h1 is a title/Header to draw your attention to a topic or idea on the page. It can be used at the top of the paragraph in order to indicate what the paragraph will be about.

An example <h1>H1 explained </h1 >

h

h2 is the second largest header tag. It is used to insert heading inside a body tag. It can be used as the heading of a paragraph. Can be used to define the title of a topic on a webpage.

E.g <h2>This is a Title</h2>
This is the text which is clearly smaller than the title.

h3

The h3 element defines a level-three heading. This heading is more important than an H4 but less important than an H2. The deprecated ALIGN attribute suggests the horizontal alignment for the heading on visual browsers. Possible values are left, right, center, and justify.

E.g. <h3>Sub header</h3>

p

The element <p> represents paragraphs in html. It is referred to as blocks of text that is separated between lines of blank space, utilizing indentation

Ex. <p> Hello I like green, Hello I like green, Hello I like green </p>

Ex. <p> Hello I like blue, Hello I like blue, Hello I like blue </p>

em

The <em>tag modifies the text by italicizing the text within the tags.

the <em>tag modifies text like this.

img

Used to insert images to an HTML webpage

E.g. <img src="killua.zoldyck.jpg" alt="Killua Zoldyck" width="300" height="400" >

<img alt="puppy" src="images/puppy.jpg" width="300" height="200">

E.g. <img src="gon.freecs.jpg" alt="Gon Freecs" width="100" height="100"> >

strong

The "strong" tag is used to indicate a content is important. The content within get emphasized. The strong element is an inline element

E.g. <strong>This text is important</strong>

HTML Comments

HTML Comments <!--...--> is used to add a comment it the source code of the website that is not displayed on the website itself. This is used to explain the code to anyone editing the source code later.

Examples: 1) <!--This is a comment-->
2) <!--missing code here-->
