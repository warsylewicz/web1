Unit 2: HTML Elements

DOCTYPE

DOCTYPE tells the browser what type of document it is.

E.g. <!DOCTYPE html>
E.g. <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

html

Root element of an HTML page and contains all other elements.

E.g. <html>

head

Container for the metadata of the page, i.e. information that is not directly part of the page. about the page.

E.g. <head>

body

Container for the contents of the page.

E.g. <body>
