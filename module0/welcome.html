Welcome to Web Development 1

This course introduces you to the fundamental concepts and technologies required for website development. You'll begin by examining how websites work and building a static webpage using basic HTML elements and logical document structure. You'll also examine cascading style sheets (CSS) and use CSS to lay out and style an HTML webpage. You'll explore the basic functionalities of JavaScript and use it to add interactivity to a website. You'll also get hands-on experience deploying a website to the internet and implementing web pages.

The topics covered in this course are:

  • Website Development Fundamentals
  • HTML (Hypertext Markup Language) Fundamentals
  • Using Sectioning Elements
  • Using Cascading Style Sheets (CSS)
  • Using Interactive HTML Elements
  • Applying CSS to Page Layout
  • Understanding JavaScript
  • Applying JavaScript to a Website
  • Publishing a Website
